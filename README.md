# MTCSync

# Overview

MTC(MIDI Time Code)を用いたライブ全体の同期システム

# Requirement

## macOS
- libmpv.so
- mido
- python-rtmidi(requirement mido)
- timecode
- questionary

## Windows
- libmpv.dll
- mido
- python-rtmidi(requirement mido)
- timecode
- questionary

# Usage

```# py(thon) mtcsync.py```

# Features

[x] MTC decode
[x] mpv link
[] load timeline
[] output MIDI
[] setting GUI
[] auto stop
